# 文字列の分割(split())

```python
test = ['orange', 'apple', 'banana', 'strawberry']
test.split(",") # test というリストを 区切り文字” , ” で区切る
test = "orange apple banana strawberry"
test.split()　# 引数に何も指定しない場合、スペースやタブ等で自動的に区切る
```

### 書式

#### スペースやタブ等で自動的に区切る
- 文字列.split()
#### 区切り文字” , ” で区切る
- 文字列.split(',')
