# Matplotlib

> データの視覚化するための機能が豊富なライブラリ
> ```python
> import matplotlib.pyplot as plt
> plt.plot(x軸に対応するデータ, y軸に対応するデータ)
> ```
> を使うだけでグラフを簡単に描画できる
> 作成したグラフは、matplotlib.pyplot.show()で表示できる。
